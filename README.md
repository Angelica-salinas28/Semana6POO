En el siguiente codigo se presenta

Un proyecto donde es una aplicación desarrollada en Python que demuestra el uso correcto de los principios de Programación Orientada a Objetos (POO).
El programa está organizado en carpetas, separando responsabilidades y siguiendo buenas prácticas de desarrollo, para que pueda ejecutarse sin problemas en Visual Studio Code.

Se implementan los conceptos fundamentales de POO como:

* Herencia
* Encapsulación
* Polimorfismo
